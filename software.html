ntHash ntHash is a hashing algorithm tuned for processing DNA/RNA sequences. It performs the best when calculating hash values for adjacent k-mers in an input sequence, operating an order of magnitude faster than the nearest competitors in typical use cases. Github link: https://github.com/bcgsc/ntHash
